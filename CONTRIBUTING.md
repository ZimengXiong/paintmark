# Contributing to Paintdown

Paintdown is an early-stage native document renderer. Changes to layout should preserve the same geometry across preview, HTML where applicable, and PDF output.

## Development

```sh
npm ci
npm run check
```

Serve the repository root with any static file server to use the interactive workbench. The demo bundle is generated by `npm run build`.

Please include a focused regression test for layout fixes. Avoid committing `dist`, generated PDFs, screenshots, or temporary render output.

## Fonts and assets

Only redistribute fonts and visual assets whose licenses permit it. Commercial fonts can be tested through the demo’s local TTF importer and should not be added to the repository.
